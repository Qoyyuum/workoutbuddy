About WorkoutBuddy

WorkoutBuddy is a cross-platform fitness companion app designed to help you track workouts, monitor nutrition, and stay motivated. Built with Flutter, it runs on Android, iOS, Windows, macOS, Linux, and the web.

  • Track your workouts and progress
  • Log food and nutrition
  • Customizable settings
  • Fun pixel art and sound effects

Get Started

Download the app from your favorite app store or run it from source. See the README for details.
