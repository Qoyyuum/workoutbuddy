How to Report Issues

If you encounter bugs, have feature requests, or want to provide feedback, please report issues directly on our GitHub Issues page.

  1. Go to the Issues tab on GitHub.
  2. Click "New Issue" and fill out the template.
  3. Describe your problem or suggestion clearly.
  4. Submit the issue and check back for updates.

Thank you for helping us improve WorkoutBuddy!
